## API ректификатора

На данный момент API представляет набор функций следующей сигнатуры:

```python
# Исправление проективного искажения
def rectify_perspective(
        # Входное изображение
        img_src: numpy.ndarray,
        
        # Контур кода на исходном изображении в обходе эталона кода
        # начиная с левого нижнего угла против часовой стрелки
        border: numpy.ndarray,
        
        # Соотношение сторон эталона в формате x_ratio к y_ratio
        x_ratio: int = 1,
        y_ratio: int = 1
)
```

```python
# Исправление согнутого угла
def rectify_bent_corner(
        # Входное изображение
        img_src: numpy.ndarray,
        
        # Контур кода на исходном изображении в обходе эталона кода
        # начиная с левого нижнего угла против часовой стрелки
        border: numpy.ndarray,
        
        # Индекс вершины согнутого угла в border
        corner_idx: int,

        # Соотношение сторон эталона в формате x_ratio к y_ratio
        x_ratio: int = 1,
        y_ratio: int = 1
)
```
